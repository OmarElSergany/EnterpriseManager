Selects basic course details.

Training Course           -        Select the course to be created.

Trainer                         -        Select the user who will run the course.

Start Date                    -        Date the course should start.

Consecutive Days         -        If ticked the course will run on consecutive days.

Exclude Saturday         -        If ticked the course will not be scheduled on a Saturday.

Exclude Sunday            -        If ticked the course will not be scheduled for a Sunday.

Adjust Staff Hours      -        If ticked then staff who are involved will automatically have their working hours adjusted to fit the course.
