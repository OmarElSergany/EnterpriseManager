You can view invoices from within the Invoice Manager or when viewing users:

The invoice view allows you to view invoice details.
Invoice Actions
If you have permission you will be allowed to change the invoice date.

Print Label                     -                  Prints a label for the shipping address.

Print Invoice                   -                  Prints a copy of the invoice

Tracking                         -                  Allows the updating of the tracking reference

User Notes                      -                  Notes the user placed when making the order

Issue Refund                    -                  Issue a full or partial refund on the invoice

Cancel                             -                  Cancel’s the invoice.

Staff Notes                     -                  Staff notes are only visible to members of staff, notes can be up to 32000 characters long.

View User                        -                  View’s the user details for the user.

Change Sales Person      -                  Allows the member of staff details, for individual items to be changed.

Close                               -                  Closes the view Invoce.
