Install Type

There are three types of installs:



Stand Alone            -     The program will be used on a single computer only.

Client                     -     Install's the ERM only.  If you have multiple computers on a network, you can install the server on a single computer, then install the client on each computer in your network.

Server                     -     Used in conjunction with Client, the server is the main computer n your network and will contain the database.
