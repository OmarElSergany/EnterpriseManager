A missing link will automatically try and redirect a missing web page on your website to an existing page.

Missing Page            -             This is the page, without the URL which could not be found.

Redirect Page          -             This is the page where the user will be redirected to.
In the above example, a user tried to visit their invoices page, which was moved to a different area of the website.
You do not need to include the domain name part of the URL
