Order Manager Settings
 Previous Top Next
These settings affect orders that are shipped via the Dispatch Orders section.

Prepared By             -             Text to specify who the order was prepared by.

Text Alignment       -             Determines where the Prepared By text is aligned on the invoice.

No Stock Dispatch   -             If ticked, users will not be allowed to dispatch any orders where there is no stock available.

Bypass Label Print  -             If ticked, the POS will not require that a label is printed prior to being dispatched.
