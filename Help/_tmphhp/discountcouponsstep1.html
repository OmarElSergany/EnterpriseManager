Step 1 allows you to define the name, start and expiry date/time.

·    Name: Unique coupon name

·    Start Date: Date/time the coupon can be used.

·    Expires: Date/time the coupon ceases to be usable.

·    Active: Weather the coupon is active (can be used) or inactive (cannot be used).

·    Free Postage: If set then the user who uses the coupon can have free postage.

·    Minimum Spend: Minimum amount of products the user must have in their shopping cart to use the voucher.

·                          Maximum usage: The maximum number of times the coupon can be used.
