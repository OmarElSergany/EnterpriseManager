System Emails
 Previous Top Next
You can configure and change the emails which are sent by the POS and website.

The system is designed to send emails at different times when triggered.  You can update how an email appears, or the language used.

Emails                      -             Type of Email.

Key                           -             Configurable options within an email, these items will be replaced with the actual value stored in settings.

Subject                     -             The subject/title of the email.

Message                   -             The message within the email.

Save                          -             Saves user changes.
