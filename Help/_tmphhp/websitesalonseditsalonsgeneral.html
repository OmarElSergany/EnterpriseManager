General settings contains contact details for the salon

Contact Name          -             Name of primary contact within the salon.

Picture                     -             This will be a link to a jpg or gif image which is 120 x 120 pixels.

Telephone                -             Telephone number for salon.

Fax                           -             Fax number for salon.

Email                        -             Email address for the salon.

Website                    -             Homepage for the salon.

Address                    -             Address of salon, without postcode.

Postcode                  -             Postcode for salon, this will be used within the website to find nearest salon.
