SMS Reminder Alerts
 Previous Top Next
SMS Reminder alert allow staff to send SMS messages to customers as a reminder of an upcoming appointment.

You can create an account with Text Magic (https://www.textmagic.com/) a 3rd party sender of SMS messages.
Sender Name.  The sender name as created on the text magic website (https://my.textmagic.com/online/reply-options/senderids).
Username.  The username for the account (https://my.textmagic.com/online/api/rest- api/keys)
API Key.  The integration API key (see above link).
