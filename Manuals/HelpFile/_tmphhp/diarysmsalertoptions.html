The diary allows users to send text notifications about upcoming appointments through the SMS Alerts option.
The homepage for Text Magic is: https://www.textmagic.com/
All settings for this option are available from the Text Magic home page.

Sender Name            -             Sender Name as provided by Text Magic.

Username                  -             Username as provided by Text Magic.

API Key                   -             API Key as provided by Text Magic.
