Continuously           -             If checked, then pool commission is paid continuously, using Wait period after an invoice is received.

Annually                  -             If checked, then the pool commission is paid annually on the Date specified.  Invoices received after the Wait period will be payable the following year.

Date                         -             If commission is paid annually, select the date at which commission is paid.

Wait                         -             Minimum number of months to wait after an invoice is received, before commission is payable.  This is to allow for cancellations and returns etc.
