Advanced diary settings.

Cache                       -             If ticked diary appointments will be saved in memory, this makes it quicker to show the appointments, but, external updates may be slower to show.

Appointment Text    -             Determines whether a customers name is shown on the first line of the appointment within the diary, or the treatment is shown.
If you have multiple sites viewing the same diary sheet, and staff at both locations can add appointments, then caching appointments could potentially allow for appointments to be booked simultaneously.  See the Replication section for more information.
