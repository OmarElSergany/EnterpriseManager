Shortcut Keys
 Previous Top Next
The following key combinations can be used as a short cut to perform certain actions within the POS.
Ctrl + C
Copies the selected text.
Ctrl + X
Cut's the selected text.
Ctrl + V
Paste's previously cut or copied text.
Ctrl + O
View orders received.
F1
Display help for current window, if available.
F8
Show user settings.
F9
Locks the POS, requiring the user to logon again
F11
Reconnects the POS to the Replication engine.
F12
Force the replication engine to replicate.
