Orders

Orders can be created via the POS using Create Order, or via a website.
Orders can have the following statuses:

·    Order Received          -    Order has been received and is waiting to be processed.

·    Order Processing   -    Order is being processed ready for dispatch.

·    Order Dispatched  -    Order has been dispatched to the customer.
All orders will have an initial status of Order Received and customers can view the status of their order online.
