If the staff member is to be included within the diary, you can edit the settings in the diary tab:

If the staff member has no diary option then click “Create Diary”, to remove the staff members diary click “Delete Dairy”.


Staff Name                 -           Name that appears within the diary.

Working Days            -           Standard working days for the staff member.

Working Hours          -           Standard working hours for the staff member.

Lunch                          -           Specify standard lunch times for the staff member.

Book Current Day     -           Allows members of the public to book an appointment via the websiteii Integration with a website requires the system to be able to integrate with a public facing website. for the current day.

Public Diary               -           Allows members of the public to book an appointment via the website.


________________________
i Integration with a website requires the system to be able to integrate with a public facing website.
