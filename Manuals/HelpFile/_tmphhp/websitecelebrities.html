You can edit the celebrity details held on the website.

Create New              -             Creates a new Celebrity endorsement for the website.

Delete                      -             Delete an existing celebrity endorsement from the website. Highlight the selected video and click delete.

Save                          -             Once changes have been made to a celebrity endorsement click save to save the changes.

Name                        -             Celebrity Name.

Description              -             Celebrity endorsement description.

Image                        -             Select from predefined images.
