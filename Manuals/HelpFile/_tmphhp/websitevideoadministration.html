There is a section on the website that can show different videos from YouTube.

Create New              -             Creates a new video for the website.

Delete                      -             Delete an existing video from the website.  Highlight the selected video and click delete.

Save                          -             Once changes have been made to a video click save to save the changes.

Description              -             Description of the video.

Video Reference      -             Unique YouTube reference.
When viewing videos on YouTube you can obtain the unique reference from the URL in your browser window.  I.e. the following link refers to the Ella Jane VIP launch:
https://www.youtube.com/watch?v=OcwfJIcQEjE
The unique reference is the section after the v= in this case it is OcwfJIcQEjE
